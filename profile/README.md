# Hidden Systems

<img src="../assets/images/hidden-systems-hero.png" alt="Hidden Systems — Illuminating World Systems" width="100%">

### Illuminating World Systems

Hidden Systems is a research-based educational project that makes the complex, often-invisible systems shaping daily life understandable to a general audience. From food production and financial markets to energy supply chains and legal structures — these systems affect everyone, yet their inner workings are rarely visible to the people inside them.

> *Understanding is the prerequisite for caring. Caring is the prerequisite for improving.*

## Repositories

### [hidden-systems](https://github.com/hidden-systems/hidden-systems)

The main project: an Astro-powered website where each topic is presented as an **interactive directed graph** — a navigable map of concepts, causes, and connections. Readers move through a system spatially, from broad overview to granular detail, at their own pace.

- **Interactive graph navigation** — explore a system the way it actually works, not as a linear document
- **Research-driven** — primary sources, cited throughout
- **Smooth transitions** — View Transitions API, GSAP animations, hexagonal overview layouts
- **Near-zero cost** — static site hosted on Cloudflare Pages; no database, no server

### [discussions](https://github.com/hidden-systems/discussions)

Public GitHub Discussions repository powering reader comments on the site via [Giscus](https://giscus.app). All discussion data lives transparently in the open.

## Philosophy

Most of the systems that govern daily life were not designed for transparency. They evolved, were optimized, were captured, and were obscured — not always deliberately, but structurally.

Hidden Systems is built on the belief that illumination is the first step toward agency.

- **Primary research** — not summaries of summaries
- **Cited sources** — every claim is traceable
- **Honest uncertainty** — where data is limited or contested, we say so
- **Editorial independence** — sponsorships fund research; they don't direct it
- **Open to correction** — factual errors, with sources, are always welcome
